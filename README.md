An edited version of the ctOS boot animation from Watch Dogs.

Properly supports password/question prompts and messages.

# Installation
To install it, download the source and either run the install.sh,
or copy the contents of images (not the folder), the ctos.plymouth and the ctos.script to /usr/share/plymouth/themes/ctos/ and install it manually.
